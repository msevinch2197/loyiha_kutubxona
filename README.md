# 📚 Library Management System (Python OOP)

## 📌 Loyihaning maqsadi

Bu loyiha Python dasturlash tilida Object-Oriented Programming (OOP) tushunchalarini o‘rganish uchun yaratilgan.

## 🧱 Tuzilishi

Loyihada 2 ta asosiy class mavjud:

### 1. Book

Kitob haqida ma’lumot saqlaydi:

* title (nomi)
* author (muallifi)
* is_available (mavjudligi)

### 2. Library

Kutubxonani boshqaradi:

* Kitob qo‘shish
* Kitob olish (borrow)
* Kitob qaytarish (return)
* Kitoblar ro‘yxatini ko‘rish

## 🚀 Ishga tushirish

1. Python o‘rnatilgan bo‘lishi kerak
2. Faylni ishga tushiring:

```bash
python main.py
```

## 📷 Natija

Dastur quyidagilarni amalga oshiradi:

* Kitoblarni ko‘rsatadi
* Kitobni olish va qaytarishni boshqaradi

## 🎯 O‘rganiladigan tushunchalar

* Class va Object
* Constructor (`__init__`)
* Methodlar
* Encapsulation

## ✨ Kengaytirish g‘oyalari

* Foydalanuvchi (User) class qo‘shish
* GUI (Tkinter) qo‘shish
* Faylga saqlash (JSON yoki TXT)
